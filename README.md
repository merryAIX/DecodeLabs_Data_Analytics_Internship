# DecodeLabs_Data_Analytics_Internship

## Overview
This repository contains the projects and technical assignments completed during the Data Analytics Industrial Training program by DecodeLabs. It serves as a portfolio demonstrating applied data analysis workflows, from initial data preparation to exploratory data analysis and beyond.

## Repository Structure

### Project 1: Data Cleaning and Preparation
* Status: Completed
* Description: Processed raw data to ensure structural integrity and quality for analysis.
* Key Techniques: Strategic imputation of missing values, duplicate removal, date standardization (ISO 8601), and general text/numerical formatting.
* Deliverables: Cleaned dataset and a formal Change Log documenting all transformations.

### Project 2: Exploratory Data Analysis (EDA)
* Status: Completed
* Description: Conducted an exploratory investigation of the dataset using Power BI to uncover sales trends, evaluate product performance, and identify anomalies.
* Key Techniques: Descriptive statistics, visual diagnostics (scatter, line, and bar charts), and trend/outlier detection.
* Deliverables: Executive Summary Report (PDF) detailing key findings and actionable business recommendations.
